# Codedhims
> A structured log of what I actually learn, not what I pretend to know.

Belajar sekaligus bereksperimen di bidang programming, dengan menekankan bahwa **All good / All okay**...karena software, sistem atau aplikasi tidak pernah dibangun berdasarkan dogma. Semuanya berdiri di atas trade-off, constraint, dan keputusan yang harus dipertanggungjawabkan. 

Software engineer bukan sekadar programmer dengan titel lebih bagus; perannya menuntut disiplin, standar kerja, dan kesadaran bahwa setiap keputusan teknis membawa konsekuensi jangka panjang. Bukan soal menulis kode lebih banyak, tapi memahami konteks, risiko, dan dampaknya terhadap sistem. Di level ini, kualitas bukan muncul dari kecerdasan, tapi dari kebiasaan: konsistensi, kejujuran teknis, dan kemampuan menolak solusi yang hanya terlihat pintar di permukaan.

> Disclaimer : pembahasan akan lebih banyak di C#, Zig, Java, Rust.
