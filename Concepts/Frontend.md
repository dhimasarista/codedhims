## Nice to Have

* **Component Model** : Function / Class / Razor / Single-file components
* **State Management** : Reactive assignments, Context API, Stores, Cascading Parameters
* **Routing** : Built-in / Router library / Modular / Decorator-based
* **Rendering** : CSR / SSR / Compile-time optimizations / WebAssembly
* **Side Effects & Lifecycle** : Hooks, lifecycle methods, reactive statements
* **Form / Validation** : Built-in validation, schema/DTO-based, class-validator equivalents
* **Security** : XSS/CSRF protection, Auth (JWT, OAuth2), role/permission control
* **API Layer** : RESTful / GraphQL / WebSocket / gRPC
* **Testing** : Unit + Integration + e2e support
* **Build / Tooling** : CLI, scaffolding, bundling, hot reload
* **Observability** : Logging, metrics, monitoring, error tracing
* **Microservices / Realtime** : Optional message brokers, real-time events, queues & jobs
